# WhisperWall Frontend

Minimal React frontend for WhisperWall - Anonymous College Feedback System.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. The frontend will run on `http://localhost:3000`

## Pages

- `/` - Submit Feedback page
- `/feed` - Public Feedback Feed
- `/admin/login` - Admin Login
- `/admin/dashboard` - Admin Dashboard

## Features

- Submit anonymous feedback
- View public feedback feed with search and filters
- Admin authentication
- Admin dashboard to manage feedback
- Add replies to feedback
- Change feedback status
- Delete feedback

## Note

This is a minimal frontend for demonstration purposes. The focus of the project is on the backend API.
